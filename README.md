# Wellmart-Technopreneur-B-Class-Project
Wellmart: Sistem Kasir &amp; Pemesanan Kantin Sekolah Berbasis Web Wellmart adalah aplikasi kasir berbasis website yang dibuat untuk mempermudah transaksi jual-beli di lingkungan sekolah (seperti kantin atau koperasi). Website ini dibangun menggunakan HTML, CSS, dan JavaScript.
