## 🚀 Fitur Website
Website ini dibuat sebagai *Single Page Application* (SPA) sederhana dengan fitur berikut:
- **Sistem Role Login:** Bisa membedakan login sebagai **Admin** (untuk kelola toko) dan **User** (untuk belanja).
- **Keranjang Belanja:** Pembeli bisa memilih jumlah barang, menambahkannya ke keranjang, dan melihat total belanja secara otomatis.
- **Diskon Spesial (Membership):** Otomatis mendeteksi nama pembeli. Jika yang login adalah Ustadz (misal: Ustadz Arif Rosadi atau Ustadz Ja'far Rais), tagihan otomatis dipotong diskon 20%.
- **Opsi Pengambilan:** Pembeli bisa memilih "Ambil di Tempat" atau "Pesan Antar (Delivery)" dengan notifikasi animasi pop-up interaktif.
- **CMS Admin (CRUD):** Admin dapat Menambah (Create), Melihat (Read), Mengedit (Update), dan Menghapus (Delete) produk secara langsung, data akan tersimpan di *Local Storage* browser.

## 💻 Teknologi yang Digunakan
Karena ini adalah *project frontend* murni tanpa server berbayar, teknologi yang digunakan adalah:
- **HTML5:** Untuk kerangka halaman web.
- **CSS3:** Untuk mempercantik tampilan (Desain modern warna Biru-Kuning).
- **JavaScript (Vanilla):** Untuk membuat logika keranjang, perhitungan kasir, dan database lokal (*Local Storage*).

## 🛠️ Cara Menjalankan Website
1. Download atau *clone* repository ini.
2. Buka folder `wellmart`.
3. Klik dua kali pada file `index.html` (akan otomatis terbuka di browser seperti Google Chrome).
4. Selesai! Kamu bisa langsung mencoba login.
   - Ketik admin untuk masuk ke halaman kelola produk.
   - Ketik nama siswa (contoh: Raihan) atau nama ustadz untuk mulai berbelanja.
